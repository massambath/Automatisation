# Automatisation

Automatisation d’une tâche rébarbative avec Selenium (login + téléchargement automatique).

## Description
Ce projet récupère automatiquement des données depuis un site web, les traite, puis envoie un email avec les résultats. Idéal pour les notifications automatisées ou le reporting.

## Fonctionnalités
- Récupération automatique de données via Selenium
- Envoi d’email avec résultats ou rapports
- Configuration facile via un fichier `config.py`
